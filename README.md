# ecommerce-Syntaxwear

`ecommerce-Syntaxwear` é um projeto front-end de vitrine digital para uma marca de moda fictícia. Ele foi criado para demonstrar um layout moderno, visual atraente e navegação simples em um site de comércio eletrônico.

## Sobre o projeto

O objetivo principal é apresentar um e-commerce de produtos com foco em usabilidade e estilo. A página inicial foi estruturada para destacar categorias, itens em promoção e oferecer um fluxo de leitura natural, do topo ao rodapé.

## Principais recursos

- Design responsivo: adapta-se bem a telas desktop, tablet e celular.
- Estrutura visual clara: header com navegação, banner principal, seções de destaques e footer informativo.
- Exposição de produtos: grade de produtos e categorias com imagens de alta qualidade.
- Uso de componentes CSS organizados: separação de estilos em arquivos específicos para facilitar manutenção.
- Conteúdo personalizado: combina identidade visual com elementos de moda e lifestyle.

## Estrutura do projeto

- `index.html` — página principal do e-commerce
- `css/reset.css` — reset básico para normalizar estilos entre navegadores
- `css/variables.css` — definições de cores, fontes e espaçamentos reutilizáveis
- `css/base.css` — estilos globais e tipografia
- `css/components/` — estilos específicos para header, hero, grades de produtos, footer e categorias
- `images/` — imagens usadas no site, incluindo banners, ícones e produtos

## Como executar

1. Abra a pasta do projeto no seu editor de código.
2. Abra `index.html` em um navegador.
3. Explore o layout e personalize o conteúdo nos arquivos HTML e CSS.

## Personalização

- Mude as cores em `css/variables.css` para adaptar a identidade da marca.
- Atualize os textos e nomes dos produtos em `index.html`.
- Substitua as imagens em `images/` para renovar o catálogo.

## Tecnologias usadas

- CSS3
